# Campus Room and Facility Booking System

## Project Title

Campus Room and Facility Booking System

---

## Problem Statement

Currently classrooms, seminar halls, labs, and meeting rooms are booked manually. This creates several problems:

* Double bookings
* Scheduling conflicts
* No centralized tracking system
* Wasted administrative time
* Poor room utilization

---

## Proposed Solution

Develop a web-based centralized booking system where users can:

* View room availability
* Book rooms online
* Cancel or modify bookings
* Receive booking confirmations

Administrators can:

* Manage rooms and facilities
* Approve requests
* Monitor room usage and analytics

---

## Users

## 1. Normal Users

Examples:

* Students
* Faculty
* Staff

They can:

* Register / Login
* View rooms
* Book rooms
* Cancel bookings
* View my bookings

---

## 2. Admin

Admins can:

* Login
* Add / Edit / Delete rooms
* Approve booking requests
* View all bookings
* Access analytics dashboard
